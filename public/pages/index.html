Liste des Exercices

Exercice 1: Long Polling
Mise en œuvre d'un gestionnaire de tâches asynchrone pour simuler des mises à jour en temps quasi-réel.
Exercice 2: Server-Sent Events (SSE)
Streaming de données unidirectionnel pour les notifications ou les flux d'informations.
Exercice 3: Chat Bidirectionnel (WebSocket)
Développement d'une application de chat simple avec une connexion persistante.
Exercice 4: Chat Multi-Salon (Socket.IO)
Extension du chat pour gérer plusieurs salons de discussion en temps réel.
Exercice 5: Notes Collaboratives
Création d'une application de prise de notes en temps réel avec édition collaborative.
